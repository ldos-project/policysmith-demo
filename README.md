# Vulcan Leaderboard - LDOS 2026

This is an archived snapshot of the Vulcan Cache Eviction Heuristic Competition leaderboard from the LDOS 2026 PhD Research School.

## About

The LDOS 2026 PhD Research School was held in Madison, WI on June 16-17, 2026. As part of the tutorial, participants developed cache eviction heuristics evaluated across 30 scenarios (10 traces × 3 cache sizes).

This static site preserves the final competition results, including:
- All submission rankings and metrics
- Complete scenario results for each submission
- Source code for Vulcan heuristics
- Interactive filtering and sorting

## Metrics

- **MRR (Mean Ratio to FIFO)**: Average performance ratio compared to FIFO baseline across all scenarios
- **Byte Hit Rate**: Percentage of requested bytes served from cache
- **Object Hit Rate**: Percentage of requested objects served from cache

## Data

The complete dataset is available in `data.json`, including all submissions, FIFO baselines, and detailed per-scenario results.
